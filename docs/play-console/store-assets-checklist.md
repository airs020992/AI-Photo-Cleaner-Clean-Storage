- App name: AI Photo Cleaner
- Short description: available in `store-listing-draft.md`
- Full description: available in `store-listing-draft.md`
- Release notes: available in `internal-test-release-notes.md`

## Required Graphics

- App icon: candidate available at `docs/play-console/assets/app-icon-512.png`.
- Feature graphic: candidate available at `docs/play-console/assets/feature-graphic-1024x500.png`.
- Phone screenshots: at least 2, recommended 4-6.
- Tablet screenshots: optional unless targeting tablets explicitly.

## Screenshot Candidate Set

Use real-device screenshots from:

`docs/play-console/screenshots/`

Recommended final order:

1. Clean dashboard with recoverable space.
2. Large videos review or dashboard card.
3. Similar photos review or scanning state.
4. Bottom navigation / settings / privacy control.

## Production Quality Bar

- Screenshots should show real product UI, not placeholder mockups.
- Text must be readable at Play Store preview size.
- Do not show private user media unless staged test media is safe to publish.
- Prefer storage outcome language: space, review, keep, remove, confirm.
- Avoid aggressive claims like "delete all duplicates automatically".
